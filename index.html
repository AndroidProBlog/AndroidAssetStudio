Android Asset Studio

Uma coleção de ferramentas para gerar facilmente recursos como ícones para seus aplicativos Android.

Gerador de Ícones de Launcher
Geração de ícones para launcher e Google Play Store.
Gerador de Ícones de Atalho
Geração de ícones de atalho para aplicativos (Android 7.1+).
Gerador de Ícones de Notificação
Geração de ícones de notificação para seu aplicativo. Eles aparecem na barra de status do sistema.
Gerador de Imagens Nine-patch
Geração de imagens nine-patch (.9.png) para o seu aplicativo, normalmente usadas para widgets personalizados de interface do usuário.
Gerador de Ícones Genéricos
Geração de ícones quadrados para uso personalizado no seu aplicativo.
Gerador de Ícones para Action Bar/Tab
Geração de ícones para a Action Bar e Tab Bar.
